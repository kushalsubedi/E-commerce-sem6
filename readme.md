## to run the project 

### clone the project ``` git clone git@github.com:kushalsubedi/E-commerce-sem6.git ```

#### follow the commands 
- for non-windows user
```make dev-init```
- for windows user 
``` make dev-init-windows ```
- install dependencies 
``` make installs ```
- build css files 
``` make build-css ```
- run server 
``` make run```
